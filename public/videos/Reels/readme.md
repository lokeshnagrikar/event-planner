# ST Events Local Reels Video Assets

Please move/copy your video files from `src/app/Reels/` to this folder:
* `Vdo1.mp4` -> `public/Reels/Vdo1.mp4`
* `Vdo2.mp4` -> `public/Reels/Vdo2.mp4`
* `Vdo3.mp4` -> `public/Reels/Vdo3.mp4`

This allows Next.js to serve them as static assets at `/Reels/[filename]`.
